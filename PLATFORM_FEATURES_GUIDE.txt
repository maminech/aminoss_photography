================================================================================
                        TESTING GUIDE
================================================================================

üîç HOW TO TEST AS ADMIN:
------------------------
1. Go to: Innov8photography.vercel.app/admin
2. Login with admin credentials
3. TEST QUICK ACTIONS:
   - Click "Create Invoice" button
   - Click "Add Expense" button
   - Click "View Bookings" button
   - Click "Manage Photobooks" button
4. TEST INVOICE PAYMENT:
   - Go to Invoices section
   - Find an unpaid invoice
   - Click green checkmark "Mark as Paid" button
   - Confirm the action
   - Verify invoice shows as "Paid"
5. TEST CURRENCY:
   - Check Dashboard - should show "TND" not "MAD"
   - Check Packages page - should show "TND" not "$"
6. TEST PHOTOBOOK MANAGEMENT:
   - Go to Photobooks section
   - Verify you can see all photobooks (including approved ones)
   - Try approving a submitted photobook


üîç HOW TO TEST AS CLIENT:
-------------------------
1. Go to: Innov8photography.vercel.app/client/login
2. Login with client credentials
3. TEST QUICK ACTIONS:
   - Click "My Galleries" card
   - Click "Guest Uploads" card
   - Click "My Photobooks" card
   - Click "Photobooth Prints" card
4. TEST PHOTOBOOTH PRINTS:
   - Click "Photobooth Prints"
   - Verify you see photobooth-generated images
   - Try downloading individual print
   - Try "Download All" button
   - Test gallery filter (if you have multiple events)
5. TEST PHOTOBOOKS:
   - Click "My Photobooks"
   - Verify you see all your photobooks
   - Check status indicators
   - Try continuing a draft (if available)
6. TEST GUEST UPLOADS:
   - Click "Guest Uploads"
   - Verify you can see photos guests uploaded
   - Try downloading guest photos

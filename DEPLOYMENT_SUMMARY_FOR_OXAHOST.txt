================================================================================
                         TROUBLESHOOTING
================================================================================

Common Issues and Solutions:
----------------------------

Issue: "Error connecting to database"
Solution:
- Check DATABASE_URL is correct in environment
- Verify MongoDB Atlas IP whitelist
- Test connection with: node -e "..."
- Add server IP to MongoDB Atlas Network Access

Issue: "502 Bad Gateway"
Solution:
- Application not running
- Check if Node.js process is active
- Restart application: pm2 restart app-name
- Check logs for errors

Issue: "Images not loading"
Solution:
- Verify Cloudinary credentials in .env
- Check next.config.js has image domains
- Test Cloudinary API access
- Check browser console for errors

Issue: "Admin login fails"
Solution:
- Verify NEXTAUTH_URL matches your domain
- Clear browser cookies
- Check NEXTAUTH_SECRET is set
- Restart server

Issue: "Email not sending"
Solution:
- Verify Gmail App Password is correct
- Check server allows port 587 (SMTP)
- Test with curl: curl -v telnet://smtp.gmail.com:587
- Check Gmail security settings

Issue: "Page not found (404)"
Solution:
- Verify build was successful
- Check .next folder exists
- Restart Node.js server
- Clear Next.js cache: rm -rf .next
